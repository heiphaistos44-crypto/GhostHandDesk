============================================
   GhostHandDesk - Application Portable
============================================

Cette application est 100% portable et ne laisse aucune trace.

UTILISATION :
  1. Double-cliquez sur ghosthanddesk-tauri.exe
  2. Le serveur HTTP se lance automatiquement sur localhost:8080
  3. Votre Device ID s'affiche dans l'interface
  4. Partagez votre ID avec l'autre personne
  5. Connectez-vous mutuellement

FICHIERS :
  - ghosthanddesk-tauri.exe : Interface et client (13 MB)
  - signaling-server.exe : Serveur de signalement (8.9 MB)

TECHNIQUE :
  - WebSocket: ws://localhost:8080/ws
  - WebRTC P2P pour la connexion directe
  - Aucun certificat TLS requis (localhost uniquement)

SUPPORT :
  - GitHub: https://github.com/Mo-101/GhostHandDesk
  - Rust + Tauri + Go + WebRTC

Aucune installation requise - Aucune trace laissée !
